# Golden Star Hotel - Sistema de Gestión Hotelera

Sistema web para la gestión de reservas y administración de un hotel ficticio.

## Funcionalidades

- Registro y autenticación de usuarios (clientes y administradores).
- CRUD de habitaciones y reservas.
- Gestión de disponibilidad y calendario de reservas.
- Panel de administración para supervisar operaciones.

## Tecnologías utilizadas

- HTML5, CSS3, JavaScript
- Backend: (PHP)
- Base de datos: (MySQL)
